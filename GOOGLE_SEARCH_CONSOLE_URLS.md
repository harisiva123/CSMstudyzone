## How to Submit to Google Search Console

### Method 1: URL Inspection Tool (Individual URLs)
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Select your property: **csmstudyzone.in**
3. Use the **URL Inspection** tool at the top
4. Enter each URL from the list above
5. Click **Request Indexing** for each URL

### Method 2: Sitemap Submission (Recommended)
1. After deployment, your sitemap will be at: **https://www.csmstudyzone.in/sitemap.xml**
2. Go to Google Search Console → **Sitemaps**
3. Submit: **https://www.csmstudyzone.in/sitemap.xml**
4. Google will automatically discover all pages with canonical URLs

### Method 3: Bulk URL Submission
1. Go to Google Search Console → **URL Inspection**
2. Use the **Bulk URL Inspection** feature (if available)
3. Paste all URLs from the list above
